# Game Asset Pipeline Imports

Generated assets from `E:\game_asset_pipeline` are copied here for Unity shell review.

These files are presentation assets only. They must not introduce authoritative simulation rules, state mutation, command resolution, or module facts inside Unity.

Suggested category use:

- `Terrain`: 2.5D sandbox terrain tiles and map texture pieces
- `Buildings`: county city, prefecture city, fort, ferry, and relay-station tokens
- `Icons`: UI icons for seal, tally, grain, money, document, and settlement markers
- `Sprites`: 2.5D sandbox unit and character reference sheets
- `Backgrounds`: menu, event, campaign, and sand-table backdrop images
- `Atlas`: packed tile atlases and JSON metadata
- `References`: test renders and calibration images

After reviewing generated art, promote only approved files into the stable Unity art folders.
